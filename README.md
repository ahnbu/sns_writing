# Advanced Writing Flow (SNS 글쓰기 에이전트)

## 📌 프로젝트 소개

이 프로젝트는 단순한 글쓰기 보조 도구를 넘어, 프로페셔널한 수준의 콘텐츠를 제작하기 위한 **에이전트 기반의 8단계 고급 집필 워크플로우**입니다.
아이디어 구상부터 심층 리서치, 집필, 비평, 그리고 최종 편집에 이르기까지, 인간 작가와 에이전트가 협업하여 최상의 결과물을 만들어내는 시스템입니다.

특히 **Perplexity MCP**를 활용한 심층 리서치와 **Notion MCP** (선택 사항) 연동을 통해 데이터에 기반한 탄탄한 글쓰기를 지원합니다.

## ✨ 8단계 핵심 워크플로우 (Core Workflow)

이 프로젝트는 다음의 체계적인 과정을 통해 글을 완성합니다:

1.  **Step 0. 초안 기록 (Original Draft)**
    - 사용자의 거친 아이디어와 메모를 있는 그대로 보존합니다.
2.  **Step 1. 논리 필터 (The Logic Filter)**
    - 글의 불필요한 '기름기'를 제거하고 단단한 논리적 뼈대를 구축합니다.
3.  **Step 2. 리서치 큐레이션 (The Insight Curator)**
    - Perplexity AI를 통해 해당 주제에 대한 최신 트렌드와 'Golden Nuggets(결정적 통찰)'을 발굴합니다.
4.  **Step 3. 페르소나 집필 (The Persona Writer)**
    - 타겟 독자를 고려한 매력적인 톤앤매너(Persona)를 입혀 초고를 작성합니다.
5.  **Step 4. 베테랑 편집 (The Veteran Editor)**
    - 모바일 환경에 최적화된 가독성 편집과 클릭을 부르는 'Hook' 제목을 설계합니다.
6.  **Step 5. 쓰레드 생성 (The Threads Generator)**
    - 완성된 글을 소셜 미디어(SNS) 환경에 맞춰 바이럴 가능한 쓰레드 시리즈로 변환합니다.
7.  **Step 6. 냉철한 비평 (The Critic)**
    - 제3자의 시각에서 원고를 냉정하게 분석하고 약점을 지적합니다.
8.  **Step 7. 최종 개선 (The Refiner)**
    - 비평가의 피드백을 반영하여 완성도를 극한으로 끌어올린 v2(수정본)를 발행합니다.
    - _(자동 수행)_ **Step 8. 히스토리 아카이브 (The History Keeper)**: 전체 작업 과정의 의사결정과 발전 과정을 요약하여 기록합니다.

## 🛠️ 수시 호출 도구 (On-Demand Tools)

정해진 프로세스와 별도로, 필요할 때 언제든 호출하여 사용할 수 있는 특수 도구들입니다.

- `/critic`: "내 아이디어 논리적인가?" -> 즉각적인 논리 검증 및 레드팀 수행
- `/insight`: "너무 뻔한 이야기 같은데?" -> 통찰력을 더해 글의 '엣지' 생성
- `/editor`: "어떻게 고쳐야 할까?" -> 문장 및 구조에 대한 편집장의 멘토링

## 📂 디렉토리 구조

```
root/
├── .agent/
│   └── skills/
│       └── advanced-writing/   # 프롬프트 및 스킬 정의 파일
├── docs/                       # 프로젝트 관련 문서 및 리서치 자료
├── output/                     # 생성된 글 저장소 (자동 분류)
│   └── {YYYYMMDD}_{Keyword}/   # 날짜_키워드 형식으로 프로젝트 폴더 생성
│       ├── 00_draft.md
│       ├── 01_logic_filter.md
│       ├── ...
│       └── 07_history_archive.md
└── README.md
```

## 🚀 시작하기

1.  **초안 입력**: 에이전트에게 쓰고 싶은 주제나 대략적인 초안을 입력하세요.
2.  **단계별 진행**: 에이전트가 안내하는 단계에 따라 진행 승인을 하거나, 수정 요청을 합니다.
3.  **결과 확인**: 각 단계가 완료될 때마다 `output` 폴더에 결과물이 자동 저장됩니다.

## ⚠️ 필수 요구 사항

- 이 시스템은 외부 정보를 실시간으로 검색하기 위해 **Perplexity MCP** 사용을 권장합니다.
