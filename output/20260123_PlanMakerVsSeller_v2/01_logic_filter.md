# Step 1. Logic Filter (논리 해체 및 구조화)

## 1. 과장 및 감정적 표현 제거 (De-Hyping)

| 원문 표현                | 문제점        | 수정 방향                   |
| ------------------------ | ------------- | --------------------------- |
| "뚝딱 만드는"            | 과장된 의성어 | "신속하게 생성하는"         |
| "휴지 조각이 될 확률"    | 자극적 비유   | "실무 적용이 어려운 결과물" |
| "뇌가 멈춘 상태"         | 과장된 표현   | "명확한 사고 없이 작성된"   |
| "훌륭한 동어반복 생성기" | 비꼬는 어조   | "핵심 없이 반복되는 결과물" |

## 2. 훈계조 문장 전환 (No Preaching)

| 원문                     | 수정                       |
| ------------------------ | -------------------------- |
| "~가 되어서는 안 됩니다" | "~의 역할이 요구됩니다"    |
| "숨지 마십시오"          | "명확한 설명이 필요합니다" |
| "~해야 합니다"           | "~가 효과적입니다"         |

## 3. 구조적 원인 분석

### 배경: 기술 발전에 따른 역할 재정의

- AI 도구의 발전으로 기획서 '제작(Making)' 자체의 진입장벽이 낮아짐
- 결과물 생성 속도가 비약적으로 향상되었으나, 결과물의 '설득력'은 여전히 인간의 영역

### 핵심 전환점

- **From:** 제작자(Maker) - 정보를 수집하고 문서를 만드는 역할
- **To:** 판매자(Seller)/편집자(Editor) - 결과물을 검증하고 핵심을 전달하는 역할

## 4. 중간 요약본 (논리적 뼈대)

### 현상

- AI 기술로 인해 기획서 제작의 물리적 비용(시간, 노력)이 급격히 감소
- 그러나 기획의 본질적 가치(설득력, 논리성)는 자동화되지 않음

### 핵심 주장

기획자에게 요구되는 역량이 '제작 능력'에서 '편집/판매 능력'으로 전환되고 있음

### 세 가지 검증 기준

1. **결과 중심 커뮤니케이션**: 과정이 아닌 핵심 가치를 즉시 전달하는 능력
2. **매체 간 일관성**: 구두 설명과 문서의 핵심 메시지가 일치해야 함
3. **본질적 이해**: 전문 용어 없이 누구나 이해할 수 있는 언어로 설명 가능해야 함

### AI 시대의 차별화 요소

1. **명확한 사고(Clear Thinking)**: AI에게 정확한 지시를 내리기 위한 전제 조건
2. **검증 역량(Verification)**: AI 결과물의 품질을 판단하고 책임지는 능력
3. **압축 능력(Compression)**: 과잉 정보를 핵심 가치로 요약하는 능력
